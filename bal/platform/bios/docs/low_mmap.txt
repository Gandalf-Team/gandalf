=== Memory map of low memory (<1MiB) ===

    0x0600 -> 0x2FFF                Stage 1
    0x3000 -> 0x6FFF                Stage 1.5
    0x7000 -> 0x7FFF                Stage 1(.5) stack

    0x8000 -> 0xBFFF                Paging structures (x86_64)
    0xC000 -> 0xFFFF                BAL stack
    0x10000 -> 0x2????              BAL
